# rabix-cliche

Rabix tool command line generator


## Install

- Clone repository
- Install node and npm
- run `npm install` in directory

## Usage

#### Base command:
    rabix-cliche

#### Required parametars:
 
    - <path-to-tool-json>

#### Example

    rabix-cliche <path-to-tool-json> <path-to-job-json>
